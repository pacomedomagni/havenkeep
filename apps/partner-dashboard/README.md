# HavenKeep Partner Dashboard

> **Status: Not yet implemented.** This will be a Next.js web app for realtors and builders to track their referrals and commissions.

## Planned Features
- Referral link generator (unique per partner)
- Dashboard: total referrals, active users, commissions earned
- Commission payouts via Stripe Connect
- Performance analytics

## Tech Stack (Planned)
- Next.js 14+ (App Router)
- Supabase Auth (partner accounts)
- Shared Supabase database
- Tailwind CSS

## Priority
This dashboard is Phase 3 of the build plan. The core mobile app (Phase 1) and receipt OCR (Phase 2) come first.
